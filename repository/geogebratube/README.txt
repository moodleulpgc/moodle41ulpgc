GeoGebraTube Repository for Moodle 2.7+

INTRODUCTION:
The GeoGebra GeoGebraTube Repository allows you to pick a material from GeoGebraTube

REQUIREMENTS:
Moodle 2.7 or above
JavaScript enabled in your browser

INSTALLATION:
- Place the "geogebratube" directory within your_moodle_install/repository/
- Visit the admin notifications page and complete the installation
- Enable the repository
- Done!

USAGE:
- Whenever you use the filepicker and an external file is a valid choice you are able to pick a material on GeoGebraTube. The
result is a link to the material.
