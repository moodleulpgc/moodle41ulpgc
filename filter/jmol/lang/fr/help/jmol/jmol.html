Mesures avec Jmol

  • Distance (2 atomes) :

    1. double-click sur le 1er atome

    2. pour un affichage permanent, double-click sur le 2ème atome

  • Angle (3 atomes) :

    1. double-click sur le premier atome

    2. click sur le 2ème atome (atome central de l'angle)

    3. pour finir double-click sur le 3eme atome

  • Angle de torsion (dièdre sur 4 atomes) :

    1. double-click sur le 1er atome

    2. click sur le 2eme atome

    3. click sur le 3eme atome

    4. pour finir, double-click sur le 4eme atome

  • Dans tous les cas:

    • glisser le pointeur sur un atome pour voir la mesure sans l'afficher de façon définitive

    • déplacer la souris hors de la fenêtre pour annuler la mesure

    • répeter la mesure pour annuler l'affichage
