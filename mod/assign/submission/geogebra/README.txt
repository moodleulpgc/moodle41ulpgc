GeoGebra assignment submission module for Moodle 2.7+

INTRODUCTION:
The GeoGebra assignment submission plugin allows teachers to set up assignments which enables students to edit a GeoGebra
Worksheet and save it in Moodle.

REQUIREMENTS:
Moodle 2.7 or above
Use of Moodle 2.3's new assignment module (mod/assign)
GeoGebra questiontype
JavaScript enabled in your browser
Modern browser

INSTALLATION:
- Place the "geogebra" directory within your_moodle_install/mod/assign/submission/
- Visit the admin notifications page and complete the installation
- Done!

USAGE (teacher):
- As a teacher, create (or edit) an assignment
- Under "Submission settings", check "GeoGebra" and select the template (or perspective) to use
- Save the assignment

USAGE (student):
- In the assignment submission, find the "GeoGebra" section
- Edit the GeoGebra worksheet
- Submit the assignment
